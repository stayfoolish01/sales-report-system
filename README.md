# 営業日報システム

営業担当者が日々の訪問活動を記録し、上長が確認・コメントできるWebアプリケーション

## プロジェクト概要

このシステムは、営業チームの日報管理を効率化するために開発されています。

### 主要機能

- **日報管理**: 営業担当者が訪問記録、課題・相談（Problem）、明日の予定（Plan）を記録
- **コメント機能**: 上長が部下の日報にコメントを追加
- **顧客マスタ管理**: 顧客情報の一元管理
- **営業マスタ管理**: 営業担当者情報と組織構造の管理
- **ダッシュボード**: 日報の提出状況や未読コメントを一覧表示
